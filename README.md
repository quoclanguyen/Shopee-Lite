# Shareandcare project

#### Key Features:
- Authentication based on JWT with RSA algorithm

#### Install packages
```
npm install
```

#### Setup .env file
```
DEV_APP_PORT=
DEV_DB_HOST=
DEV_DB_PORT=
DEV_DB_NAME=

PRO_APP_PORT=
PRO_DB_HOST=
PRO_DB_PORT=
PRO_DB_NAME=

NODE_ENV=
```

#### Start the app
```
npm run dev
```
